# 大文档：智能计算系统课程手册（节选合成）

本文档用于模拟单文件超长文本场景，便于测试分块与检索。

## 第一章 课程目标

智能计算系统课程旨在介绍从算法到硬件加速的完整链路。学生应掌握向量检索、近似最近邻、以及在大规模语料上的评估方法。

### 1.1 向量检索基础

向量检索将文本编码为稠密向量，通过内积或余弦相似度寻找近邻。常见索引包括 HNSW、IVF 与乘积量化。评测指标常用 Recall@k 与 MRR。

### 1.2 近似最近邻

当数据规模超过百万条时，精确 kNN 代价过高，需采用 ANN。权衡在于召回率与查询延迟；工业系统常采用分层索引与缓存热查询。

## 第二章 RAG 架构

检索增强生成将外部知识注入大模型。典型流程为：分块、嵌入、向量库存储、查询时检索 Top-K、拼上下文再生成。

### 2.1 多文档场景

多文档时需为每个片段记录来源文件名与章节标题，以便答案可引用出处并降低跨文档噪声。混合检索（稀疏+稠密）在专有名词密集时往往更稳。

### 2.2 大文档场景

大文档的核心矛盾是上下文窗口有限。策略包括：语义分块、重叠窗口、层级摘要与子块检索、以及检索后重排序。

## 第三章 实验与指标

课程期末项目要求提交代码、PPT 与实验报告。报告需描述实现过程、展示 Demo 截图，并明确指出相对基线的改进。

### 3.1 自建问答对

若无官方标注，可自建 10–20 条问答对，用于对比不同分块与检索策略下的命中率。

### 3.2 系统指标

除答案质量外，应记录索引构建时间、单次查询延迟与首 token 时间（若调用在线大模型）。
