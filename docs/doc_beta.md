# Beta 实验平台

## 概述

Beta 是面向开发者的在线实验平台，提供 A/B 测试、特性开关、用户分群等功能。目前处于公测阶段（Open Beta），所有注册用户均可免费使用基础功能。

## API 限流

Beta API 采用令牌桶算法进行限流，具体参数如下：

- **每密钥每分钟请求上限：60 次**
- **突发令牌桶容量：120**
- 超出限流后返回 HTTP 429 状态码，Retry-After 头指示重试等待秒数

简单来说，Beta API 限流规则是：每个 API 密钥每分钟最多可以发送 60 个请求。如果短时间内突发流量很大，令牌桶最多可以容纳 120 个请求（即可以短时间内发出最多 120 个请求，但随后必须等待令牌补充）。

开发者可以通过响应头 X-RateLimit-Remaining 和 X-RateLimit-Reset 来监控剩余配额。

## 数据留存

用户上传的实验数据默认保留 30 天，可在设置中延长至 90 天。数据到期后自动进入回收站，回收站内数据保留 7 天后彻底删除。

## 与其他产品关系

Beta 与 Alpha 的计费账户互通，但 API 密钥相互独立，不可混用。也就是说，你可以在 Alpha 的账单中看到 Beta 的消费记录，但调用 Beta API 时必须使用 Beta 专属的 API 密钥。

## 定价

公测期间免费。正式发布后按调用次数计费，预计定价为每千次调用 0.5 元。正式定价将在公测结束前 30 天公布。
